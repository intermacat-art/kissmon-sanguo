# 親親獸剪紙三國

HTML5 Canvas Survivor-like 小遊戲。親親獸一人對抗三國，自動射劍氣，撿經驗升級選技能，打死 BOSS 過關。

## 玩法

- **WASD / 方向鍵** 移動
- **SPACE** 釋放奧義（累積罐頭後）
- 自動發射劍氣攻擊最近敵人
- 撿藍球 = 經驗值（升級選強化）
- 撿 ♥ = 補血  ·  撿 🥤 = 累積奧義罐
- 擊敗關底 BOSS = 通關
